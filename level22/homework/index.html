# 1) 0-დან მომხმარებლის რიცხვამდე დაპრინტვა num = int(input("შეიყვანეთ რიცხვი: ")) for i in range(0, num + 1): print(i) print("----------------------------------") # 2) 0-დან მომხმარებლის რიცხვამდე ჯამის პოვნა num = int(input("შეიყვანეთ რიცხვი: ")) sum = 0 for i in range(0, num + 1): sum += i print("რიცხვების ჯამი არის:", sum) print("----------------------------------") # 3) 3-ის გამოტოვებით რიცხვების დაბეჭდვა num = int(input("შეიყვანეთ რიცხვი: ")) for i in range(0, num + 1): if i == 3: continue print(i) print("----------------------------------") # 4) 4-ის გამოტოვებით ნამრავლის პოვნა num = int(input("შეიყვანეთ რიცხვი: ")) product = 1 for i in range(0, num + 1): if i == 4: continue if i != 0: product *= i print("ნამრავლი არის:", product) print("----------------------------------") # 5) BOSS LEVEL: ორი რიცხვს შორის ლუწების ჯამი start = int(input("შეიყვანეთ პირველი რიცხვი: ")) end = int(input("შეიყვანეთ მეორე რიცხვი: ")) sum_even = 0 for i in range(start, end + 1): if i % 2 == 0: sum_even += i print("ლუწი რიცხვების ჯამი არის:", sum_even)
